DIGtv – MTV-style channels

Each channel interleaves music videos from one set of sources with station bumpers from another, then writes a native playlist you can press Play on from any device (web, TV, Roku, etc.). Edit a channel, click Save all, then Regenerate to (re)build its playlist.


Nightly rebuilds run automatically (Dashboard → Scheduled Tasks → “DIGtv – Regenerate channels”). Data is mirrored to plugins/DIGtv/channels.json.
